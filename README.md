## Topics Covered

1. **Teknik Integrasi** — substitution, integration by parts (LIATE), trigonometric integrals, trigonometric substitution, partial fractions
2. **Aplikasi Integral** — area between curves, arc length, volume (disk, washer, shell)
3. **Kekonvergenan Deret** — convergence tests (divergence, p-series, integral, comparison, limit comparison, alternating series, ratio, root), absolute vs. conditional convergence
4. **Deret Maclaurin & Taylor** — expansions, important series table
5. **Tabel Rumus Cepat** — antiderivatives, integration by parts results
6. **Prediksi Pola Soal UAS** — question distribution and topic breakdown
